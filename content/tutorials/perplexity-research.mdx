# Perplexity 研究指南：AI 搜尋引擎的正確用法

Perplexity 不只是 ChatGPT 的替代品，它是一個**AI 搜尋引擎**。最大特色是每個回答都附上來源引用，讓你可以驗證資訊的正確性。

## Perplexity vs ChatGPT

| 功能 | Perplexity | ChatGPT |
|-----|------------|---------|
| 即時搜尋 | ✅ 預設開啟 | 需手動開啟 |
| 來源引用 | ✅ 每句都有 | 偶爾提供 |
| 資訊時效 | 即時 | 有訓練截止日期 |
| 學術研究 | 專門優化 | 一般用途 |

## 步驟 1：開始使用

1. 前往 [perplexity.ai](https://perplexity.ai)
2. 不需註冊即可使用基本功能
3. 註冊後可保存對話歷史

## 步驟 2：基本搜尋

直接輸入問題，Perplexity 會：
1. 搜尋多個網站
2. 整合資訊
3. 生成回答
4. 附上來源連結

**範例搜尋**：
```
2024 年最熱門的 AI 工具有哪些？
```

回答會包含 [1]、[2]、[3] 等標註，點擊可查看來源。

## 步驟 3：使用 Pro Search

Pro Search 是進階搜尋模式，會：
- 詢問釐清問題
- 搜尋更多來源
- 給出更深入的分析

**如何開啟**：
1. 在搜尋框旁邊開啟「Pro」開關
2. 免費用戶每天有 5 次額度

## 步驟 4：Focus 模式

選擇特定來源類型：

- **All**：搜尋所有網站
- **Academic**：只搜尋學術論文
- **Writing**：幫你寫作
- **YouTube**：搜尋影片內容
- **Reddit**：搜尋 Reddit 討論

**範例**：用 Academic 模式搜尋論文
```
最新的 Transformer 架構研究進展
```

## 步驟 5：上傳檔案

可以上傳 PDF、圖片等檔案讓 Perplexity 分析：

1. 點擊搜尋框的附件圖示
2. 上傳檔案
3. 詢問相關問題

**範例用法**：
- 上傳論文 PDF，請 AI 摘要重點
- 上傳圖片，詢問圖中內容

## 步驟 6：建立 Collection

Collection 是專題研究空間：

1. 點擊側邊欄「Collections」
2. 建立新的 Collection，例如「AI 研究」
3. 相關搜尋都存在這個空間

好處：
- AI 會記住這個專題的上下文
- 方便整理研究資料

## 實用搜尋技巧

### 1. 要求特定格式
```
用表格比較 ChatGPT、Claude、Gemini 的優缺點
```

### 2. 限定時間範圍
```
2024 年 AI 產業的重大事件
```

### 3. 追問深入
Perplexity 支援多輪對話，可以追問：
```
更詳細說明第三點
```

### 4. 要求引用學術來源
```
請引用學術論文說明 RAG 的原理
```

## 適合用 Perplexity 的情境

1. **事實查核**：需要有來源的準確資訊
2. **學術研究**：找論文和研究資料
3. **新聞追蹤**：了解最新事件
4. **競品分析**：研究市場和競爭對手
5. **技術學習**：查詢技術文件和教學

## 不適合用 Perplexity 的情境

1. **創意寫作**：ChatGPT 更適合
2. **程式開發**：用 Cursor 或 Claude
3. **長對話**：Claude 更擅長
