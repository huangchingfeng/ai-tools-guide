# Midjourney 提示詞技巧大全

Midjourney 是目前最受歡迎的 AI 圖像生成工具之一，而好的提示詞是生成高品質圖片的關鍵。

## 基礎語法

Midjourney 的基本提示詞格式：

```
/imagine prompt: [主題描述], [風格], [參數]
```

例如：
```
/imagine prompt: a cute cat sitting on a windowsill, watercolor style, --ar 16:9
```

## 提示詞結構

一個完整的提示詞通常包含：

1. **主題** - 你想畫什麼
2. **細節** - 顏色、材質、環境
3. **風格** - 藝術風格、攝影風格
4. **參數** - 比例、品質等

## 常用參數

### 圖片比例 --ar

```
--ar 1:1    # 正方形（預設）
--ar 16:9   # 橫向寬螢幕
--ar 9:16   # 直向手機螢幕
--ar 4:3    # 傳統螢幕比例
--ar 3:2    # 相片比例
```

### 風格化程度 --stylize (--s)

```
--s 0      # 最忠於提示詞
--s 100    # 預設值
--s 1000   # 最高藝術化
```

### 品質 --quality (--q)

```
--q 0.25   # 快速生成，品質較低
--q 0.5    # 一般品質
--q 1      # 高品質（預設）
```

### 混亂度 --chaos

```
--chaos 0    # 結果一致
--chaos 100  # 結果多變
```

## 進階技巧

### 權重調整

使用 `::` 來調整關鍵詞權重：

```
cat::2 dog::1  # 貓的權重是狗的兩倍
```

### 負面提示詞

使用 `--no` 來排除不想要的元素：

```
beautiful landscape --no people, buildings
```

### 圖片參考

上傳圖片作為參考：

```
/imagine prompt: [圖片URL] a portrait in similar style
```

### 多重提示詞

使用 `::` 分隔不同概念：

```
hot:: cold::  # 同時包含熱和冷的概念
```

## 常見問題

**Q: 為什麼生成的圖片和我想的不一樣？**

可能原因：
- 提示詞不夠具體
- 關鍵詞順序影響權重
- 試著增加更多細節描述

**Q: 如何讓人物的手看起來正常？**

這是 AI 的已知限制，可以嘗試：
- 避免特寫手部
- 使用 `--v 6` 最新版本
- 多生成幾次挑選
