# 用 Claude 寫程式的完整指南

Claude 是一個非常強大的程式開發助手，本教學將教你如何有效地使用 Claude 來提升開發效率。

## 為什麼選擇 Claude？

Claude 在程式開發方面有幾個優勢：

- **長文本理解**：可以處理整個檔案或多個檔案
- **清晰的解釋**：不只給程式碼，還會解釋為什麼這樣寫
- **Artifacts 功能**：可以即時預覽程式碼結果
- **較少幻覺**：相比其他模型，更少產生錯誤程式碼

## 基本用法

### 1. 請求寫程式

```
用 Python 寫一個函式，輸入一個數字列表，回傳其中所有偶數的平方和
```

Claude 會給你完整的程式碼並解釋邏輯。

### 2. 解釋程式碼

```
請解釋這段程式碼在做什麼：
[貼上程式碼]
```

### 3. 除錯

```
這段程式碼出現錯誤：[錯誤訊息]
程式碼如下：
[貼上程式碼]
```

## 進階技巧

### 給予完整上下文

不好的提示：
```
幫我寫一個登入功能
```

好的提示：
```
我正在用 Next.js 14 + TypeScript 開發一個電商網站。
請幫我寫一個登入功能：
- 使用 NextAuth.js
- 支援 Google 和 Email 登入
- 登入後跳轉到 /dashboard
```

### 使用 Artifacts 預覽

當你要求 Claude 寫前端程式碼時，可以請它使用 Artifacts：

```
用 React + Tailwind CSS 做一個計算機元件，
請用 Artifacts 讓我可以直接預覽
```

### 請求測試程式碼

```
為上面的函式寫單元測試，使用 Jest
```

| API 設計 | 設計一個 RESTful API 來管理使用者資料 |

## 搭配 Claude Code 使用

Claude Code 是 Anthropic 推出的 CLI 工具，可以直接在終端機中使用 Claude：

```bash
# 安裝
npm install -g @anthropic-ai/claude-code

# 使用
claude
```

優點：
- 直接讀取專案檔案
- 可以編輯和建立檔案
- 執行終端機命令
- 完整的開發流程整合

## 注意事項

1. **驗證程式碼**：AI 產生的程式碼可能有 bug，務必測試
2. **安全性**：不要讓 AI 處理敏感資料（密碼、API Key 等）
3. **著作權**：注意程式碼的授權問題
4. **持續學習**：把 AI 當作學習工具，理解它給的程式碼
