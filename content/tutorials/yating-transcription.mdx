# 雅婷逐字稿使用教學

雅婷逐字稿是由 Taiwan AI Labs（台灣人工智慧實驗室）開發的語音轉文字工具，專為繁體中文優化。

## 為什麼選擇雅婷逐字稿？

- **繁體中文最準確**：專為台灣語音訓練
- **支援台語**：獨家支援台語辨識
- **在地用語**：認識台灣特有詞彙
- **免費使用**：基本功能完全免費

## 第一步：開始使用

### 網頁版

1. 前往 [asr.yating.tw](https://asr.yating.tw)
2. 不需要註冊，直接上傳音檔
3. 等待轉錄完成

### 手機 App

1. 在 App Store / Google Play 搜尋「雅婷逐字稿」
2. 下載安裝後開啟
3. 可即時錄音轉文字

## 第二步：上傳音檔

支援的格式：
- MP3
- WAV
- M4A
- 其他常見音訊格式

操作步驟：
1. 點擊「選擇檔案」
2. 上傳你的音檔
3. 選擇語言（中文/台語）
4. 點擊開始轉錄

## 第三步：取得逐字稿

轉錄完成後你可以：

- 線上瀏覽文字內容
- 複製文字到其他地方
- 下載為文字檔

## 進階技巧

### 提升辨識準確度

1. **使用好的麥克風**：音質越好，辨識越準
2. **減少背景噪音**：安靜環境錄音
3. **清晰發音**：說話速度適中
4. **單一說話者**：多人同時說話會降低準確度

### 搭配其他工具

錄音 → 雅婷轉文字 → ChatGPT 整理重點

```
請幫我把這份會議逐字稿整理成：
1. 會議重點摘要
2. 待辦事項清單
3. 決議事項

[貼上逐字稿內容]
```

## 常見問題

**Q: 免費版有什麼限制？**

免費版每次可轉錄約 10 分鐘，每日有使用次數限制。

**Q: 辨識準確度如何？**

繁體中文準確度約 90-95%，音質好的情況下更高。

**Q: 可以辨識多人對話嗎？**

可以，但無法自動區分說話者。需要後續手動標記。

**Q: 支援即時轉錄嗎？**

App 版支援邊錄邊轉，適合做會議即時記錄。

## 小技巧

1. 重要會議建議用錄音筆，音質更好
2. 轉錄後用 ChatGPT 或 Claude 做摘要整理
3. 長時間錄音可分段上傳，效率更高
