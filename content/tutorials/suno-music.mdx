# Suno AI 音樂創作入門：用文字創作歌曲

Suno 是目前最強大的 AI 音樂生成工具，只要輸入文字描述，就能生成包含人聲、歌詞、伴奏的完整歌曲。

## Suno 能做什麼？

- 生成完整歌曲（含人聲）
- 自動創作歌詞
- 支援各種音樂風格
- 一次生成多個版本

## 步驟 1：註冊帳號

1. 前往 [suno.com](https://suno.com)
2. 用 Google、Discord 或 Apple 帳號登入
3. 免費用戶每天有 50 個 credits

## 步驟 2：簡單創作模式

最簡單的用法是直接描述你要的歌：

1. 點擊「Create」
2. 在輸入框輸入描述
3. 點擊「Create」生成

**範例描述**：
```
一首關於程式設計師加班的搖滾歌曲，歌詞要幽默
```

Suno 會自動：
- 生成適合的歌詞
- 選擇搖滾風格編曲
- 加入人聲演唱
- 產生兩個不同版本

## 步驟 3：自訂歌詞模式

想要自己寫歌詞？開啟「Custom Mode」：

1. 切換到 Custom 模式
2. 在「Lyrics」欄位輸入歌詞
3. 在「Style of Music」輸入風格
4. 點擊 Create

**歌詞格式範例**：
```
[Verse 1]
每天早上九點到公司
打開電腦開始寫 code
Bug 一個接一個出現
修到晚上十二點

[Chorus]
喔～程式設計師的生活
就是這麼 rock and roll
```

**風格描述範例**：
```
pop rock, male vocal, energetic, 120 bpm
```

## 步驟 4：風格描述技巧

### 基本元素
- **曲風**：pop, rock, jazz, hip-hop, electronic, classical
- **人聲**：male vocal, female vocal, duet, no vocal
- **情緒**：happy, sad, energetic, calm, dramatic
- **速度**：slow, medium, fast, 120 bpm

### 組合範例

**流行情歌**：
```
mandopop ballad, female vocal, emotional, piano, strings
```

**電子舞曲**：
```
EDM, electronic, synth, no vocal, upbeat, 128 bpm
```

**說唱**：
```
hip-hop, rap, male vocal, trap beat, bass heavy
```

**日系動漫風**：
```
j-pop, anime opening, female vocal, energetic, guitar
```

## 步驟 5：延長歌曲

Suno 預設生成約 1-2 分鐘的片段，想要完整歌曲：

1. 找到喜歡的版本
2. 點擊「...」選單
3. 選擇「Extend」
4. 可以選擇從哪裡延續

## 步驟 6：下載與分享

1. 點擊歌曲右側的「...」
2. 選擇「Download」
3. 可下載 MP3 或影片格式

## 進階技巧

### 1. 使用結構標記

在歌詞中使用標記控制結構：
- `[Verse]` 主歌
- `[Chorus]` 副歌
- `[Bridge]` 橋段
- `[Intro]` 前奏
- `[Outro]` 結尾
- `[Instrumental]` 純音樂段落

### 2. 控制人聲風格

在風格描述中加入：
- `raspy voice` 沙啞聲音
- `soft voice` 柔和聲音
- `powerful voice` 有力量的聲音
- `whisper` 耳語

### 3. 生成純音樂

不想要人聲？在風格中加入：
```
instrumental, no vocal
```

## 常見問題

**Q: 生成的歌曲可以上傳到 YouTube 嗎？**
A: Pro 版可以，免費版不行。

**Q: 可以指定歌手聲音嗎？**
A: 不能直接指定，但可以用風格描述來接近。

**Q: 中文歌詞效果如何？**
A: 可以用中文歌詞，但英文效果較穩定。
