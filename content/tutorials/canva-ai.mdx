# Canva AI 設計入門教學

Canva 是全球最受歡迎的線上設計工具，近年整合了多項 AI 功能，讓設計變得更簡單。

## Canva AI 功能總覽

- **Magic Design**：輸入文字自動生成設計
- **Magic Write**：AI 文案撰寫
- **Magic Eraser**：一鍵去除圖片元素
- **Magic Edit**：用文字描述修改圖片
- **Text to Image**：文字生成圖片
- **Magic Expand**：AI 擴展圖片範圍

## 第一步：註冊帳號

1. 前往 [canva.com](https://canva.com)
2. 使用 Email 或 Google 帳號註冊
3. 免費版即可使用部分 AI 功能

## 功能一：Magic Design

讓 AI 幫你自動設計！

### 操作步驟

1. 點擊「Create a design」
2. 選擇「Magic Design」
3. 上傳一張圖片，或輸入設計主題
4. AI 會生成多種設計供你選擇

### 使用範例

```
咖啡廳週年慶促銷海報，溫馨風格
```

AI 會根據描述生成多種海報設計。

## 功能二：Magic Write

AI 幫你寫文案！

### 操作步驟

1. 在設計中新增文字框
2. 點擊文字框後選擇「Magic Write」
3. 輸入你想要的文案類型

### 提示詞範例

```
幫我寫一段 Instagram 貼文，推廣我們的新品手沖咖啡，風格活潑年輕
```

## 功能三：Magic Eraser

一鍵去除圖片中不要的東西。

### 操作步驟

1. 選擇一張圖片
2. 點擊「Edit image」→「Magic Eraser」
3. 用筆刷塗抹要移除的部分
4. AI 會自動填補移除的區域

### 適用場景

- 移除照片中的路人
- 清除背景雜物
- 刪除浮水印（請尊重版權）

## 功能四：Text to Image

用文字描述生成圖片。

### 操作步驟

1. 點擊左側「Elements」
2. 選擇「AI Image Generator」
3. 輸入描述文字
4. 選擇喜歡的風格

### 提示詞範例

```
一杯熱騰騰的拿鐵咖啡，木質桌面，溫暖的早晨光線，攝影風格
```

## 功能五：Magic Expand

AI 擴展圖片邊界。

### 操作步驟

1. 選擇一張圖片
2. 點擊「Edit image」→「Magic Expand」
3. 拖曳邊界到想要的大小
4. AI 會自動補上擴展區域的內容

### 適用場景

- 把直幅照片變成橫幅
- 為設計預留更多空間
- 修復裁切過度的照片

## 實用設計場景

| 場景 | 推薦功能 |
|------|----------|
| IG 貼文 | Magic Design + Magic Write |
| 產品圖 | Text to Image + Magic Eraser |
| 簡報製作 | Magic Design + Magic Write |
| 海報設計 | Magic Design + Magic Expand |
| 大頭貼 | Magic Eraser (去背) |

## 常見問題

**Q: AI 生成的圖片可以商用嗎？**

可以！Canva 的 AI 圖片允許商業使用。

**Q: Magic Write 支援中文嗎？**

支援，但英文效果通常更好，可以先用英文生成再翻譯。

**Q: 免費版的 AI 功能夠用嗎？**

一般使用夠用，如果是大量製作內容建議升級 Pro。

## 設計小技巧

1. **善用模板**：先選模板再用 AI 調整
2. **保持一致性**：用相同的顏色和字體
3. **少即是多**：不要塞太多元素
4. **留白很重要**：讓設計有呼吸空間
