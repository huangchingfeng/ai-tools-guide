# Claude Artifacts 視覺化教學

Artifacts 是 Claude 最強大的功能之一，可以即時生成並預覽程式碼、圖表、互動式網頁。

## 什麼是 Artifacts？

Artifacts 是 Claude 對話視窗旁邊的一個獨立面板，可以：

- 即時預覽程式碼執行結果
- 生成互動式圖表
- 建立簡單的網頁應用
- 製作 SVG 圖像

## 第一步：啟用 Artifacts

1. 前往 [claude.ai](https://claude.ai)
2. 登入你的帳號
3. Artifacts 功能預設開啟，直接使用即可

## 第二步：生成你的第一個 Artifact

嘗試輸入這個提示詞：

```
幫我做一個番茄鐘計時器，有開始、暫停、重設按鈕
```

Claude 會在右側生成一個可互動的計時器！

## Artifacts 可以做什麼？

### 1. 互動式工具

```
做一個 BMI 計算器，輸入身高體重後顯示結果和健康建議
```

### 2. 數據視覺化

```
用這些數據做一個圓餅圖：
- 蘋果 30%
- 香蕉 25%
- 橘子 20%
- 其他 25%
```

### 3. 簡單遊戲

```
做一個貪吃蛇遊戲，用方向鍵控制
```

### 4. 學習工具

```
做一個日文五十音的記憶卡片，點擊顯示讀音
```

### 5. 網頁原型

```
幫我做一個咖啡廳網站的首頁原型，要有 Hero 區塊和菜單區
```

## 進階技巧

### 迭代修改

生成後可以要求修改：

```
把計時器的顏色改成綠色，並加上音效提醒
```

### 下載程式碼

點擊 Artifact 右上角的「Copy」按鈕，可以複製完整程式碼。

### 全螢幕預覽

點擊「Expand」可以全螢幕查看效果。

## 實用範例

| 類型 | 提示詞範例 |
|------|-----------|
| 工具類 | 做一個匯率計算器，支援台幣、美金、日圓 |
| 圖表類 | 用長條圖呈現這些銷售數據... |
| 遊戲類 | 做一個井字遊戲，可以和電腦對戰 |

| 設計類 | 用 SVG 畫一隻可愛的貓咪 |

## Artifacts 技術限制

- 無法存取外部 API（除非是 CORS 允許的）
- 無法儲存資料到伺服器
- 較複雜的應用可能需要拆分功能

## 常見問題

**Q: Artifacts 生成的程式碼可以商用嗎？**

可以，你可以複製程式碼用於自己的專案。

**Q: 為什麼有時候 Artifact 沒有顯示？**

確認你的提示詞有明確要求視覺化輸出，例如「幫我做」、「生成一個」。

**Q: 可以匯出成獨立的 HTML 檔案嗎？**

可以！複製程式碼後存成 .html 檔案，在瀏覽器中開啟即可。
